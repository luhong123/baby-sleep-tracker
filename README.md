# 宝宝睡眠记录应用

一个简洁的移动端网页应用，用于记录宝宝的每日睡眠情况。

## 功能特点

- 📅 记录每天的入睡和醒来时间
- ⏱️ 自动计算睡眠时长（支持跨天）
- 📝 可添加备注信息
- 💾 数据本地存储，无需服务器
- 📱 移动端优化界面
- 🎨 简洁优雅的设计

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 填写日期、入睡时间、醒来时间
3. 可选添加备注（如"睡得很香"）
4. 点击"保存记录"按钮
5. 所有记录会按时间倒序显示

## 技术说明

- 纯前端实现，无需安装依赖
- 使用 localStorage 本地存储数据
- 响应式设计，适配手机屏幕
- 支持跨天睡眠时间计算

## 文件结构

```
baby-sleep-tracker/
```

## 注意事项

- 数据存储在浏览器本地，清除浏览器数据会丢失记录
- 建议定期备份重要数据
- 如果醒来时间早于入睡时间，系统会自动判断为跨天
