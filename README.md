# 宝宝睡眠记录应用

一个移动端 PWA 网页应用，用于记录宝宝每天的睡眠情况，支持统计分析、趋势图表和数据管理。

## 功能特点

- 📅 记录每日入睡和醒来时间，自动计算睡眠时长（支持跨天）
- 🌙☀️ 区分**夜间睡眠**和**白天小睡**，分别统计
- ⚡ 一键快速记录入睡/醒来时间
- 📊 睡眠统计面板：总记录数、平均时长、最长睡眠
- 📈 趋势图表：7/14/30天睡眠时长折线图（Chart.js）
- 📅 本周概览：本周天数、平均睡眠、夜间睡眠
- ✏️ 支持编辑和删除已有记录
- ↩️ 删除/清空操作可撤销，防止误操作
- 📦 数据导出/导入（JSON 格式），支持去重合并
- 💾 数据存储在浏览器本地，无需服务器
- 📱 移动端适配，Apple 设计风格
- 📲 PWA 支持，可添加到主屏幕，离线可用

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 点击"记录入睡"快速填入当前时间，或手动填写
3. 选择睡眠类型（夜间睡眠 / 白天小睡）
4. 可选添加备注（如"睡得很香"）
5. 点击"保存记录"
6. 查看统计面板和趋势图表了解睡眠规律
7. 可通过"导出"备份数据，通过"导入"恢复

## 技术说明

- 纯前端实现，无需安装依赖
- 使用 `localStorage` 本地存储数据
- Chart.js 4.x 绘制趋势图表（CDN 引入）
- Service Worker 实现离线缓存（Cache-First 策略）
- 响应式设计，适配手机、平板屏幕
- 支持跨天睡眠时间自动计算
- PWA manifest 支持添加到主屏幕

## 注意事项

- 数据存储在浏览器本地，清除浏览器数据会丢失记录
- 建议定期使用"导出"功能备份数据
- 入睡/醒来时间相同时，系统会自动判断为跨天
- 日期不能选择未来日期
